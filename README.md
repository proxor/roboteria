# Roboteria Website

Diese Webseite ist fuer GitHub Pages mit Jekyll vorbereitet.

## Lokales Preview

1. Ruby und Bundler installieren
2. Im Projektordner:
   - `bundle install`
   - `bundle exec jekyll serve`
3. Die Seite ist dann unter http://localhost:4000 erreichbar.

## Inhalte pflegen

- Kurse: Daten in `_data/courses.yml`
- Texte/Seiten: `index.md`, `courses.md`, `projects.md`, `info.md`
- Styles: `assets/css/style.css`
